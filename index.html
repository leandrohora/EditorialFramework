O Editorial Framework é uma ferramenta de front-end que possibilita um desenvolvimento rápido e intuitivo de sites editoriais com uma gama de elementos de inteface e um sistema de grid responsivo.

Começando com Editorial Framework

Para começar de forma rápida, basta adicionar o Editorial Framework já compilado em seu projeto, fazendo o download ou utilizando o link em CDN. Abaixo seguem mais opções de download e instruções de implementação.

Estrutura de arquivos

A seguir temos um exemplo da forma mais simples de implementação da estrutura de arquivos para se iniciar um novo projeto.

Template inicial HTML

Copie e cole o arquivo de estilo <link> dentro da sua <head> antes de todos os outros arquivos de estilo para carregar o CSS do o Editorial Framework:

<link rel="stylesheet" href="https://cdn.rawgit.com/leandrohora/EditorialFramework/e56c7b94/css/editorial-framework.css">

Ou se preferir o CSS minificado:

<link rel="stylesheet" href="https://cdn.rawgit.com/leandrohora/EditorialFramework/e065caf9/css/editorial-framework.min.css">

É importante a utilização da meta tag viewport para garantir o funcionamento responsivo do sistema de grid. Com isso, a página deve ficar da seguinte forma:

<!DOCTYPE html>
<html lang="pt-br">
  <head>
    <title>Olá, mundo!</title>
    <!-- Meta tags Obrigatórias -->
    <meta charset="utf-8">
    <meta name="viewport" content="width>=device-width, initial-scale=1">
    <!-- Editorial Framework CSS -->
    <link rel="stylesheet" href="https://cdn.rawgit.com/leandrohora/EditorialFramework/e065caf9/css/editorial-framework.min.css">
  </head>
  <body>
    <h1>Olá, mundo!</h1>
  </body>
</html>
